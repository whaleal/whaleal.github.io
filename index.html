icefrog

icefrog是一个小而全的Java工具类库，通过静态方法封装，降低相关API的学习成本，提高工作效率，使Java拥有函数式语言般的优雅，让Java语言也可以“甜甜的”

event

这是一个 Java 基础类库，用于促进事件驱动编程。 该框架的目的是提供一种简单的方法来生成和处理事件。 鼓励仅使用事件作为应用程序进程和第三方系统之间的通信方式

event->

tuple

Tuple是一种数据结构，可存放多个元素，每个元素的数据类型可不同。Tuple与List集合类似，但是不同的是，List集合只能存储一种数据类型，而Tuple可存储多种数据类型。
